Bash Tutorial

## Description

Purpose of project is to learn how to write and execute simple 'Hello, World!' script on bash, python, and R. Also, practice creating conda environment, launching Jupyter Lab, and pushing files to my personal Git repository.

## Requirements

- Miniconda or Anaconda
- Python 3.10
- JupyterLab and Kernels (jupyterlab ipykernel)
- R anf R Kernel (r-base r-irkernel)

## Set Up

1. Clone repository:

```bash
   git clone https://github.com/nguyen3693/bash_env_assignment.git
```

2. Create conda environment from environment file

```bash
   conda env create -f environment.yml
```

3. Activate the environment:

```bash
   conda activate 7030_class_1
```

## Usage 

Run the script:

```bash
   python hello.py
```
or

```bash
   ./hello.py 
   ./hello.R
   ./hello.sh
```
